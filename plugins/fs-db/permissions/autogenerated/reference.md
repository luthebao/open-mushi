## Default Permission

Default permissions for the fs-db plugin

#### This default permission set includes the following:

- `allow-load-session-content`
- `allow-load-session-transcript`
- `allow-load-session-enhanced-notes`
- `allow-save-session-content`
- `allow-save-session-transcript`
- `allow-save-session-enhanced-note`
